## About

This project is a responsive website created during a JavaScript workshop. It demonstrates best practices in front-end web development, focusing on clean design, accessibility, and adaptability across devices. The project serves as a template or starting point for modern startup websites.

---

## Features

- Responsive layout for desktop, tablet, and mobile
- Modern landing page design
- Interactive navigation menu
- Section-based content (e.g., About, Services, Contact)
- Clean, semantic HTML structure
- CSS for styling and responsive breakpoints
- JavaScript for interactivity (e.g., menu toggling, smooth scrolling)

---

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/dizheleva/SoftUni-Courses.git
    cd SoftUni-Courses/JavaScript/32-34.Workshop/ResponsiveStartupWebsite
    ```
2. No dependencies are required—this is a static site.

---

## Usage

- Open `index.html` in your web browser to view the website.
- Use the responsive navigation menu to explore different sections.
- Customize content, styles, and scripts as needed for your own startup.

---

## Technologies Used

- **HTML5** – Semantic markup
- **CSS3** – Responsive design, media queries, modern layouts
- **JavaScript (vanilla)** – Basic interactivity and dynamic elements

## Project Structure

```
ResponsiveStartupWebsite/
├── index.html
├── style.css
├── script.js
└── assets/
    └── images/
```

- `index.html` – Main HTML file
- `style.css` – Styles for layout and responsiveness
- `script.js` – JavaScript for navigation and UI effects
- `assets/images/` – Images and icons used in the site
